1. Overview of the GPA Calculator System**

The **AU Calc Pro Semester GPA Calculator** is a web-based academic tool designed to help students easily calculate their semester GPA based on their obtained grades. The system provides a simple and intuitive interface where users can enter the total number of subjects and specify how many subjects fall under each grade category. The application automatically computes the GPA using the standard grade point system, making it quick and convenient for students to evaluate their academic performance.

---

### **2. User Interface and Design**

The interface of the GPA calculator is designed with a modern and user-friendly layout. The webpage includes a navigation bar with options such as **Home**, **GPA Calculator**, and **CGPA Calculator** for easy navigation. The main section contains input fields where students can enter the number of subjects and distribute them across different grade categories such as **O, A+, A, B+, B, and C**. ### **3. GPA Calculation Process**

The GPA calculation process is straightforward and efficient. Students first enter the **total number of subjects** they have taken in the semester. Then they specify how many subjects correspond to each grade category. Once the data is entered, the system processes the grade points and calculates the **Grade Point Average (GPA)** automatically. The calculated GPA is displayed visually in a circular result indicator, allowing students to instantly view their academic score.

---

### **4. Result Visualization**

The result section provides a clear visual representation of the calculated GPA. The GPA value is displayed inside a circular progress indicator, making it easy for students to quickly understand their results. Additionally, the interface includes options to **download the result as a PDF** or **share the result**, enabling students to save or share their academic performance easily.

---

### **5. Benefits of the System**

This GPA calculator simplifies the process of academic grade evaluation for students. Instead of manually calculating grade points, students can instantly obtain their GPA with minimal effort. The system saves time, reduces calculation errors, and provides a convenient digital solution for tracking academic progress.
